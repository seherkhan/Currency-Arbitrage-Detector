To run program, double click CurrencyArbitrage_DS.jar

Note: This program reads csv files which are located in the /data folder.
To load new data either:
1) click "Fetch" button after running the .jar file. This will close the program and the scraper script will run (will use Firefox browser). Will run for around 10 min. Restart .jar file after this script finishes.
2) go to /data and run forex_scrapper.exe
3) to insert sample data directly, go to /data and paste your csv file, make sure to follow the format.
